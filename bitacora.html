Proceso de desarrollo

Bitácora del Proyecto

Registro de decisiones, dificultades, cambios e implementación del TP1. Esta sección queda preparada para ser ampliada en el TP2 con nuevas reflexiones y mejoras del proyecto.

1

Planificación inicial

Se analizaron los requerimientos del trabajo práctico y se definió una estructura simple, clara y funcional: una portada principal, dos páginas individuales, una bitácora y un archivo README. También se decidió utilizar una estética moderna, con tarjetas, sombras suaves, bordes redondeados y una paleta azul/violeta para mantener coherencia visual.

2

Decisiones de diseño

El equipo eligió un diseño responsive con navegación fija superior, botones visibles y tarjetas individuales para facilitar la lectura. Se usaron tipografías de Google Fonts: Montserrat para títulos e Inter para textos. Para preservar privacidad, se utilizaron avatares ilustrados en formato SVG en lugar de fotografías personales reales.

3

Dificultades encontradas

Una dificultad fue asegurar que el menú y las tarjetas funcionaran correctamente en diferentes tamaños de pantalla. Se resolvió mediante media queries específicas para 1200px, 900px y 400px. También se cuidó que todas las páginas tengan navegación interna para no depender del botón “Atrás” del navegador.

4

Implementación con JavaScript

Se implementaron interacciones dinámicas: apertura/cierre del menú responsive, mensaje dinámico en la portada, despliegue de intereses en cada tarjeta individual y cambio visual de color en la tarjeta de presentación. Estas funciones están documentadas en el README del proyecto.

5

Cambios importantes

Se reforzó el contenido de las tarjetas, se personalizaron los perfiles de Jonatan Emanuel Uribio y Marisol Alcaraz, se mejoró la estética general del sitio y se preparó la documentación para completar con los links de GitHub, Vercel y capturas de pantalla solicitadas por la consigna.
